# Archived design specs

Design specs and analyses for deriva-ml work streams that have
landed in code or been folded into the canonical engineer audits
under `docs/audits/`. Kept as historical context for *why* the code
is shaped the way it is.

These are NOT the current contract for any subsystem.

**For current references see:**

- `docs/user-guide/` — user-facing contracts and worked examples.
- `docs/audits/2026-05-22-engineer-audit-*.md` — the canonical
  per-subsystem audits used by deriva-ml development. They cite
  these archived specs where relevant.
- `docs/adr/` — architectural decision records.
- Active specs in this directory's parent
  (`docs/superpowers/specs/`) — anything still referenced from
  source-code docstrings.

**Policy:** move specs here once they've been superseded; do not
delete. Update this README to record the "now see" pointer for
each archived stream.
